# System Pulse
# The Feeling Bible

Date: 1 July 2026

## First Three Seconds

System Pulse should make Vanessa feel calm, in control, reassured, and ready to keep creating.

That feeling is the product contract.

## Core Sentence

Stop designing System Pulse like software. Start designing it like a trusted companion that quietly checks in.

## Product Shape

System Pulse has two surfaces:

1. The Companion
2. Today's Plan

The Companion always opens first.

Today's Plan opens only when Vanessa asks for more.

## Product One: The Companion

The Companion answers:

1. Am I okay?
2. How long have I got?
3. Anything I should know?
4. Need more?

It should feel like Candle Pulse and Apple Battery: warm, quiet, confident, and almost invisible.

The Companion should show:

- Large circular heart and score
- A calm health headline
- Calm state sentence
- Estimated uninterrupted work time
- Tiny status checks for Applications, Memory, Storage, and Battery
- Open Today

Visual reference:

- Compact dark Apple-menu-style popover
- Small top notch
- Score ring on the left
- Reassurance copy on the right
- "At a Glance" status list below
- Quiet full-width Open Today button

The Companion should not show:

- Restart buttons
- Next best step
- Diagnostics
- Version number
- Live/local labels
- Refresh controls
- Explanatory paragraphs

## Product Two: Today's Plan

Today's Plan is not a dashboard.

It is an inbox for the few things worth knowing.

Today's Plan should show:

- Greeting
- "You're good for" time remaining
- Things worth knowing
- Do I need to do anything?

It should not show category cards for Flow, Browser, Applications, Memory, Storage, or Next Best Step.

If everything is fine, most things should disappear.

If one thing needs attention, one thing should appear.

## Detail Rule

Detail lives behind a click.

If Vanessa clicks Chrome, she sees Chrome.

If she clicks Storage, she sees Storage.

Do not explain everything before she asks.

## Action Rule

Never show an action unless it actually works.

A quiet action is better than a loud one.

Use labels like:

- Restart Chrome
- Later
- Ignore

Avoid labels like:

- Restart now
- Remind me
- Ignore today

The decision is not urgent. The user is.

## Taste Rules

- Calm over clever.
- Glanceable over comprehensive.
- Reassurance before diagnosis.
- One decision at a time.
- Apple Battery, not Activity Monitor.
- Candle Pulse, not a dashboard.
- White space instead of more cards.
- Confidence instead of explanation.

Everything that does not contribute to calm, control, reassurance, or creative momentum probably does not belong on screen.

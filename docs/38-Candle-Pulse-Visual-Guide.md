# Candle Pulse Visual Guide

Date: 1 July 2026

Source: Vanessa provided `Candle Pulse VIsuals.png` as the visual guide.

The PNG itself is not committed by default because it is about 1.5 MB. This lightweight note preserves the durable visual decisions for future chats.

## Core Direction

System Pulse should feel like Candle Pulse:

- Quick glance. Stay in flow.
- Full picture. Clear actions.
- Minimal. Reassuring.
- One breath check-in.

The visual style should feel calm, focused, and private. It should never feel like Activity Monitor, a technical dashboard, or a productivity nag.

## Visual Values

- Calm first: reassures before it informs.
- Focus respecting: shows what matters and never interrupts.
- Actionable: one clear suggestion at a time.
- Private by design: everything feels local, contained, and trustworthy.

## Menu Bar View

The menu bar surface is the closest reference for the Companion.

Use:

- Dark translucent rounded panel.
- Soft green glow around the pulse mark.
- Whole, clean pulse shapes. No pie-chart wedges, cut-out slices, or conic progress shadows inside the heart or score mark.
- Heart and score as the emotional anchor.
- Warm greeting beside the score.
- Plain reassurance, such as "You're good to keep working."
- Time remaining in bright green.
- Simple status rows with small symbols, dots, and quiet status labels.
- One Open Today button at the bottom.

The menu bar view should be readable in under three seconds.

## Today View

The Today surface can borrow the expanded view's atmosphere, not its heaviness.

Use:

- Dark glass window feel.
- Score and greeting at the top.
- Estimated uninterrupted work time as the primary readout.
- Things worth knowing as a small list.
- One recommendation panel only when action is genuinely useful.
- A calm reassurance strip when everything else looks good.

Avoid:

- Making cards compete for attention.
- Showing category cards just because the data exists.
- Explaining the whole computer before Vanessa asks.
- Multiple recommendations at once.

## Important Tension

The visual guide includes an expanded Today mock-up with status cards and a recommendation card.

The current product decision still stands:

- The Companion is the product.
- Today is an inbox.
- Detail lives behind a click.
- Everything that does not require attention should disappear.

Use the image for visual language, spacing, contrast, glow, typography weight, and emotional tone. Do not use it as permission to rebuild a dashboard.

## Color And Tone

Use a small, quiet palette:

- Near-black and charcoal background.
- White text for confidence and clarity.
- Candle green for healthy, time, and reassurance.
- Amber only for real attention.
- Thin low-contrast borders.
- Soft glows, never decorative blobs.

The screen should feel lit from the pulse, not decorated.

## Copy Rules

Prefer:

- "Good evening, Vanessa."
- "You're good to keep working."
- "You have 1h 50m of uninterrupted work time."
- "Things worth knowing"
- "Everything else looks good."

Avoid:

- Technical category explanations.
- Developer labels.
- Loud urgency.
- Long diagnostic paragraphs.

## Action Rules

If the Companion ever shows a suggestion, it must be:

- Real.
- Working.
- Quiet.
- One at a time.
- Easy to defer.

Do not add a suggestion card to the Companion automatically. The current Companion requirement still says no restart button or next-best-step card unless Vanessa explicitly chooses to adopt that part of the guide.

## Final Test

Before shipping a visual change, ask:

1. Can Vanessa understand it in one breath?
2. Does it reassure before it informs?
3. Is there only one decision?
4. Would this help her keep creating?
